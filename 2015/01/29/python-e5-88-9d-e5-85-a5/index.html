Python 初入

Posted on 2015-01-29   |   In Python   |  

其实我不太清楚 Python 具体有一些什么优点，带着一点玩的心态就学习了下。

当看到官网的 The State of Python (2 vs 3) 我就觉得被坑了，到底是使用2还是3，

官方竟然是这样写的 ：

Use Python 3 if...

You don’t care.
You love Python 3.
You are indifferent towards 2 vs 3.
You don’t know which one to use.
You embrace change.
Use Python 2 if...

You love Python 2 and are saddened by the future being Python 3.
The stability requirements of your software would be improved by a language and runtime that never changes.
Software that you depend on requires it.
看完这个我也没在意到底用哪个，使用的 ubuntu 自带有 python 2.7,就直接用了。 粗略看了下基本语法，接下来完成一个**用户注册和登陆**的功能。   * * * * **基本框架&工具** [Flask ](http://dormousehole.readthedocs.org/en/latest/ "Flask") 小辣椒，微型网络开发框架 [Peewee](http://peewee.readthedocs.org/en/latest/ "Peewee") ORM MySQL 这个估计没人不知道 [Psycopg2](http://initd.org/psycopg/ "Psycopg2") 数据库接口 Mysql-workbench 数据建模工具   * **环境搭建** 1> 建立虚拟环境 Virtualenv 
$ virtualenv pythonenv
cd pythonenv/bin
chmod +x activate
. activate
pip install flask
pip install peewee
2&gt; 安装 psycopg2    （[详细请看](http://stackoverflow.com/questions/5420789/how-to-install-psycopg2-with-pip-on-python/5450183)）

sudo apt-get install libpq-dev python-dev
3&gt; 安装mysql和mysql-workbench

到此建立了一个 pythonenv 独立的虚拟环境，就可以进入项目开发了。

  • 可以安静的写代码了

    在此之前，最好先试试自己的环境是否搭建好，Flask 的一个小例子可以试一下（here）。

    随意路径下建立自己的 project（文件夹），名称就是 MyBlog ，新建一个 user.py 。

    1> 导入需要用的包

from flask import Flask,request,render_template
 from flask import json
 from model import *

app = Flask(name)

2> 定义服务端返回方式

def makeReponse( ret, reason, kwarg ):
 res = { ‘ret’ : ret, ‘reason’: reason }
 if( kwarg ):
 res.update( kwarg )
 return json.jsonify( **res )

3> 定义注册方法和登陆方法

# regist
@app.route(‘/user/regist’, methods=(‘post’,))
def registUser():
print( request.data )
req = json.loads( request.data )
account = req[‘account’]
password = req[‘pwd’]
try:
res = User.create( account=account, password = password, sex=1 )
userId = res.get_id()
return makeReponse( 0, ‘’, userId=userId )
except Exception,e:
print( str(e) )
return makeReponse( -1, str(e) )

# login
@app.route(‘/user/login’,methods=(‘POST’,) )
def loginUser():
req = json.loads( request.data )
acc = req[“account”]
pw = req[“pwd”]
try:
for info in User.select().where( User.account == acc ):
if( info.password <> pw ):
raise Exception(‘invalid password ~’)
return makeReponse( 0,’’,token=acc,userId=info.get_id() )
raise Exception(‘user not found ‘)
except Exception ,e:
print( str(e) )
return makeReponse(-1,str(e))

到此，就写完了~
