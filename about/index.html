项目经历

2015: 龙视发布—电视大数据展示

龙视发布是展示观看电视节目用户数据的平台，经过大数据系统分析，在手机上展示用户操作数据。个人完成数据图表等模块，包括折线图，饼状图，map 图等，使用 echart 通过 webview 展示 map 图，其他使用 MPAndroid 图表展示。

2015: 中国移动内部资产管理客户端

中国移动内部资产管理系统手机客户端，个人完成内部资产调拨，盘查，新增模块。主要功能有条形码 扫􏰀，资产表格展示，资产地图定位，资产拍照上传等功能。

2015: 美食社交应用—萌约

此项目是基于LBS的美食社交应用，通过查看附近的人然后约吃饭。个人负责百度地图地理位置的选择， 美食日记模块和版本管理，熟悉了 LBS 使用开发，xmpp 即时通讯搭建，图片压缩和相册图片缓存。

2014: Android POS 机支付应用

在支付应用部门，负责 Android 手机，平板的银联交易支付应用开发，POS 机程序调试和移植。封装 Android EMV 内核库。期间熟悉了 Android 平台的蓝牙通讯，了解了硬件设备的通讯，NDK 开发，银联业务的规范，参与了 Android POS 的银联直联入网认证。

2014: Github 参与 Android L Meterial Design 翻译

Google I/O 大会之后􏰁出了全新设计规范 Meterial Design。在 GitHub 上参与对 Meterial Design 设计文档翻译和校对。

2013:校园代课助手—课代表

V1.0 版:由我发起，并协调团队 3 个人的开发，用了 2 个月时间完成第一版并且上线。个人负责 android 客户端和 php 服务端开发。通过这个应用，大学生可以发布代课需求和帮助同学上课拿到报酬。

V1.2 版:收集部分同学的反馈，再花 1 个月更新课程发布，push，搭建用户聊天功能等。在没有花钱的 推广下，获得 4000+多下载量。因为推广方面遇到较大阻力，项目已下架。
